## 📖 Detailed Overview

### 📚 Computer Networks (CN)

This section covers the complete CN syllabus with a focus on interview readiness.

#### 🎯 Key Topics
- **Fundamentals**: OSI & TCP/IP models, physical layer, encoding
- **Data Link Layer**: Framing, error detection/correction, flow control, MAC addressing
- **Network Layer**: IP addressing (IPv4/IPv6), subnetting, routing algorithms (Dijkstra, Bellman-Ford), congestion control
- **Transport Layer**: TCP (flow/congestion control, 3-way handshake), UDP, ports
- **Application Layer**: DNS, HTTP, FTP, SMTP, POP3, IMAP
- **Security**: Firewalls, IPSec, TLS/SSL, VPNs, cryptography basics
- **Modern Networking**: SDN, cloud networking, NFV, 5G, IoT

#### 💡 Highlights
- Visual diagrams for TCP handshake, routing algorithms, and layering models
- Formula cheat sheet and interview-specific mnemonics
- Practical implementations using `Wireshark`, `Scapy`, `Socket` programming
- Comparison tables for all major protocols and algorithms

### 💻 Operating Systems (OS)

A comprehensive guide to OS concepts with practical examples and interview strategies.

#### 🎯 Key Topics
- **Fundamentals**: Processes, threads, IPC, system calls, microkernel vs monolithic
- **CPU Scheduling**: FCFS, SJF, SRTF, Priority, Round Robin, MLFQ, Linux CFS
- **Synchronization**: Race conditions, mutexes, semaphores, monitors, Peterson's algorithm, dining philosophers
- **Deadlocks**: Conditions, RAG, Banker's algorithm, prevention, avoidance, detection
- **Memory Management**: Paging, segmentation, TLB, fragmentation (internal/external)
- **Virtual Memory**: Demand paging, page faults, replacement algorithms (FIFO, LRU, Optimal, Clock)
- **File Systems & Disk**: Contiguous/linked/indexed allocation, directory structures, disk scheduling (FCFS, SSTF, SCAN, C-SCAN)
- **Linux**: Kernel architecture, process management, system calls

#### 💡 Highlights
- Step-by-step address translation examples
- Deadlock detection algorithm visualizations
- Semaphore implementation patterns
- Practical Linux labs (process management, file operations)
- Revision cheat sheets and common interview mistakes
